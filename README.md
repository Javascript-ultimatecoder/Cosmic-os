# Cosmic-os
COSMIC OPERATING SYSTEM - Procedural god engine with particle physics

## Running the dashboard

```bash
python cosmic_dashboard.py
```

The app serves an absolute-limit cosmic FastAPI dashboard on `http://127.0.0.1:8080` with 320 gods across 7 rarity classes, a denser galaxy/nebula presentation, a live Mooore counter, Face ID status, and audit-backed intelligence upgrades.
